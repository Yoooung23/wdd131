Flow Generator Site Plan

Logan Young

WDD 131-02

Overview

Purpose

I'm going to build a tool to help generate yoga flow ideas for my wife's business.

Audience

Any yoga instructor, but mostly my wife

Dynamic elements

This site will first ask for criteria data from the user. Using that form data, such as how long you want the yoga session to go, and how advanced of a practice it will be, we'll have our javascript load objects from my object array of yoga poses and criteria, then sort them, and deliver a generasted yoga flow using the object details.

Home page

Flow

This is going to be a minimalist design, lots of space for page elements to breath. but there will be a generate button, and filter buttons. then when you have a generated flow, it will be displayed as a card with a image, a title, and small details, just enough of what you need for building a flow
